二、部署到云端

1、账号准备

发布到云端之前需要有一个 GitHub 账号或者 Gitee（码云）账号，没有的话到官网申请一个便是，二者并没有太大的区别。
主要区别在于 GitHub 服务器在国外因为众所周知的原因在国内访问速度相对较慢，而且不支持免费创建私人仓库，而码云原依托于开源中国访问速度快而且可以免费创建私人仓库，最重要的一点是它还支持中文（坏笑）。二者在使用上基本一致，到底使用哪一个，请自行斟酌。
